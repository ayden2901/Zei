### 🖇 VPS Deployment !

- ganti semua isi config.py sama punya lu kalo udah lanjut dibawah
- clone repo : `git clone https://git_token@github.com/username/nama_repo && cd nama_repo`
- Setup Vps (Jalankan hanya sekali) : `bash setup.sh`
- `tmux new-session -s ubot`
- `python3 -m venv venv && source venv/bin/activate`
- `pip3 install requirements.txt`
- `bash start`
- Klik CTRL B + D di keyboard VPS

___

### 🖇 Cara buka cek log userbot di VPS

- Ketik `tmux a -t ubot`

___

<div align="center"><h3>NOTE</h3></div>

⚠️ Kalo ada eror "sqlite3.OperationalError: unable to open database file" buka vps terus ketik :

- `tmux a -t ubot`
- Klik CTRL + C
- Lanjut `bash start`
- Close VPS

___

⚠️ Kalo divps gunicorn nya belom connect atau log divps kaya gini "Retrying in 1 seconds" ketik :

- `pkill -f gunicorn`
- `bash start.sh`
